# 🚀 Project Name (smothpol)

[![Platform: Fabric](https://img.shields.io/badge/Platform-Fabric-lightgrey)](https://fabricmc.net/)
[![Minecraft Version: 1.21.x](https://img.shields.io/badge/Minecraft-1.21.x-brightgreen)]()

An advanced optimization mod for **Fabric 1.21.x** engineered to seamlessly support massive, game-changing plugins while featuring some of the absolute best rendering fixes available for Minecraft.

---

## 🔥 Features

* **Massive Plugin Support:** Optimizes game performance to handle huge, resource-heavy plugins without breaking a sweat.
* **Next-Gen Rendering Fixes:** Packed with industry-leading solutions to eliminate common Minecraft rendering issues, stutters, and visual bugs.
* **Maximum Stability:** Enhances overall engine performance for a much smoother gameplay experience.

---

## ⚠️ CRITICAL INSTALLATION NOTE

> [!CAUTION]
> **This mod MUST be installed on BOTH the Server and the Client (Player).**
> Failing to install it on both sides will cause the server to repeatedly crash. 

---

## 🛠️ Requirements

* **Minecraft Version:** 1.21 / 1.21.1+
* **Mod Loader:** Fabric
